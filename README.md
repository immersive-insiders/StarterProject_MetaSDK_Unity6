# StarterProject_MetaSDK_Unity6
A baseline repository for VR/MR development using Meta SDK

Repository Details:
- Unity version [6000.3.9f1 LTS](https://unity.com/releases/editor/qa/lts-releases)
- Meta XR SDK
    - [Meta XR Core SDK](https://assetstore.unity.com/packages/tools/integration/meta-xr-core-sdk-269169)
    - [Meta Interaction SDK](https://assetstore.unity.com/packages/tools/integration/meta-xr-interaction-sdk-265014)

Project details:
- The project uses Universal Render Pipline.
- The Build settings have already been configured such that the built app can run on the Meta Quest devices.

NOTE:
Make sure to change the Company Name and Product Name under player settings.
